# Basiques

## ?ping

{% tabs %}
{% tab title="Description" %}
Récupérer le temps de latence du bot
{% endtab %}

{% tab title="Format" %}
```text
?ping
```
{% endtab %}
{% endtabs %}

## ?support

{% tabs %}
{% tab title="Description" %}
Obtenir les liens utiles du bot, tel que le lien du serveur discord ou le lien d'ajout du bot
{% endtab %}

{% tab title="Format" %}
```text
?support
```
{% endtab %}
{% endtabs %}

## ?invite

{% tabs %}
{% tab title="Description" %}
{% hint style="warning" %}
Cette commande est prévue pour une prochaine version
{% endhint %}

Cette commande est un alias de la commande ?support

{% hint style="info" %}
Voir aussi: ?support
{% endhint %}
{% endtab %}

{% tab title="Format" %}
```text
?invite
```
{% endtab %}
{% endtabs %}

## ?help

{% tabs %}
{% tab title="Description" %}
Envoie la page d'aide du bot \(liste des commandes\) dans vos messages-privés
{% endtab %}

{% tab title="Format" %}
```text
?help
```
{% endtab %}
{% endtabs %}

## ?user

{% tabs %}
{% tab title="Description" %}
Récupérer les informations d'un utilisateur
{% endtab %}

{% tab title="Format" %}
```markup
?user <@user>
```
{% endtab %}

{% tab title="Exemple" %}
![Exemple de la commande ?user @AL3X\_69](../.gitbook/assets/user.png)
{% endtab %}
{% endtabs %}
